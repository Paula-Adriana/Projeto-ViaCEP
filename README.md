## Projeto ViaCEP
Projeto desenvolvido no curso +Devs2Blu.
A aplicação consulta a API ViaCEP (<https://viacep.com.br>) e retorna os dados de logradouro ao usuário. Desenvolvido em Angular com Bootstrap.
